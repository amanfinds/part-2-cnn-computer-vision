What is Convolution?

Convolution is a mathematical operation used by CNNs to detect important features such as edges, textures, and patterns from images.

Why is Pooling Used?

Pooling reduces image dimensions and computational complexity while preserving important features.

Why is ReLU Commonly Used?

ReLU introduces non-linearity and helps the model learn complex patterns efficiently.

Why are CNNs Better for Images?

CNNs automatically learn visual features and spatial patterns from images, making them much more effective than regular feed-forward neural networks.
